## 📦 Установка и запуск проекта  

### 1️⃣ Установите зависимости  
> ```bash
> npm install
> ```

### 2️⃣ Запустите сервер с JSON Server  
> ```bash
> npx json-server src/shared/database/db.json
> ```

### 3️⃣ Запустите приложение  
> ```bash
> npm run dev
> ```
